# ⚡ ScanSpark

ScanSpark is a beautiful, minimalist, and lightweight QR code generator web application. Built with raw performance and modern UI/UX principles in mind, it allows users to convert any text or URL into a clean QR code instantaneously.

## ✨ Features

- **Instant Generation**: Converts URLs or plaintext to high-fidelity QR codes on the fly.
- **Custom Sizing**: Choose from small, medium, or large dimensions tailored for web or print layouts.
- **Clean UI/UX**: Designed with a modern, focus-driven interface using modern CSS and the Inter font family.
- **Direct Downloads**: Seamlessly download generated assets directly to your local drive.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Custom Variables, Flexbox), Vanilla JavaScript (ES6+)
- **API**: [GoQR API](https://goqr.me/api/)

## 🚀 Quick Start

1. Clone or download this repository.
2. Open `index.html` directly in any modern web browser.
3. Type a URL, select your size, and spark your QR code!
